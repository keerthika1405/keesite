1. What is SQL?
SQL, or Structured Query Language, is a domain-specific language used for managing and manipulating relational databases. It provides a standardized way to interact with relational database management systems (RDBMS) and is widely used for tasks such as querying data, updating data, inserting data, and deleting data in a database.

SQL, or Structured Query Language, is a domain-specific language used for managing and manipulating relational databases. It provides a standardized way to interact with relational database management systems (RDBMS) and is widely used for tasks such as querying data, updating data, inserting data, and deleting data in a database.

Chamberlin and Boyce's work aimed to create a universal language for database management. In 1979, the first official standard for SQL, SQL-79, was published by the American National Standards Institute (ANSI). This marked the beginning of SQL as a standardized language across different database systems. Subsequent versions, such as SQL-86, SQL-89, and SQL-92, continued to refine and expand the standard.

2. What are the different SQL data types?

Integer

  • INT (Integer)
  • SMALLINT (Small Integer)
  • BIGINT (Big Integer)
  • DECIMAL or NUMERIC (Exact numeric with a specified precision and scale)
  • FLOAT (Floating-point number)
  • REAL (Single-precision floating-point number)
  • DOUBLE or DOUBLE PRECISION (Double-precision floating-point number)

Character

  • CHAR(n) (Fixed-length character string)
  • VARCHAR(n) or VARCHAR2(n) (Variable-length character string)
  • TEXT (Variable-length character string, often used for large text )

Date and Time

  • DATE (Date)
  • TIME (Time of day)
  • DATETIME or TIMESTAMP (Date and time together)

Boolean

  • BOOLEAN or BOOL

Binary

  • BLOB (Binary Large Object, used for storing large binary data)
  • BINARY(n) (Fixed-length binary string)
  • VARBINARY(n) (Variable-length binary string)

Miscellaneous

  • ENUM (Enumeration of values)
  • SET (A set of values chosen from a predefined list)
  • JSON (JSON-formatted data)

Spatial

Some databases, particularly those that support spatial data, have types like POINT, LINESTRING, POLYGON for representing geometric and geographic data.
